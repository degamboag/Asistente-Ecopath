# EwE Advisor 🌊

**Asistente inteligente paso a paso para modelado trófico con Ecopath with Ecosim (EwE).**

Este proyecto es una aplicación web interactiva diseñada en **español** para funcionar como compañero en tiempo real mientras trabajas en la consola de Ecopath.

## 🚀 Características principales

- **Chat interactivo**: Realiza consultas sobre balanceo, ecuaciones y mejores prácticas del modelado trófico.
- **Análisis de capturas de pantalla**: Toma capturas de tus errores o matrices en Ecopath, pégalas directamente con `Ctrl+V` y el asistente las interpretará.
- **Extractor de parámetros desde PDF**: Sube un artículo científico en PDF y la IA extraerá de forma estructurada los parámetros tróficos (Biomasa, P/B, Q/B, EE, TL, Dieta) directamente en una tabla descargable en formato **CSV**.
- **Panel de Referencia Rápida**:
  - Guía paso a paso para construir modelos.
  - Glosario interactivo de parámetros clave (EE, P/B, Q/B, etc.).
  - Checklist de balanceo ecológico del modelo.

## 🛠️ Tecnologías utilizadas

- HTML5, CSS3 (Diseño oscuro oceánico con Glassmorphic UI).
- JavaScript Vanilla (completamente del lado del cliente, sin servidores intermediarios).
- **Gemini API** integrada (Soporte nativo multimodal para imágenes y documentos PDF).

## 💻 Instalación y uso local

1. Clona este repositorio o descarga los archivos.
2. Abre el archivo `index.html` en cualquier navegador web moderno.
3. Consigue una API Key gratuita en [Google AI Studio](https://aistudio.google.com/apikey).
4. Introduce tu clave en la app haciendo click en el botón de **Configuración (⚙️)** y ¡listo!
